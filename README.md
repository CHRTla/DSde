# Домашнее задание #3

## Sakhno
git init -инициализация репозитория
git clone <URL репозитория> - Клонирование репозитория
git add <файлы> - Добавление изменений в индекс
git commit -m "Ваш комментарий к изменениям" - Сохранение изменений
git status - Просмотр изменений
git log - Просмотр истории коммитов
git branch <название ветки> - Создание новой ветки
git checkout - Переключение на ветку
git pull origin - Обновление локальной ветки из удаленного репозитория
git push origin - Отправка изменений на удаленный репозиторий
git merge <название ветки> - Объединение изменений из другой ветки
git branch - Узнать все ветки в репозитории
